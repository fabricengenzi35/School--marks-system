EPAZA School Marks System

📚 Overview

EPAZA School Marks System is a web application designed to help schools manage student marks easily.
Teachers can enter marks, and the system automatically calculates totals, averages, percentages, and class rankings.

This system helps reduce manual calculations and makes it easier to prepare report cards and proclamation lists.

---

🚀 Features

- 🔐 Teacher and admin login
- 🧑‍🎓 Student marks entry
- 📊 Automatic total and average calculation
- 🏆 Automatic class ranking (1st, 2nd, 3rd, etc.)
- 📄 Printable report cards
- 📱 Mobile-friendly interface
- ☁️ Online hosting support

▶️ How to Use

1. Open the system in a browser.
2. Login as a teacher or administrator.
3. Enter student marks for each subject.
4. Click Calculate Results.
5. The system automatically generates totals, averages, and positions.
